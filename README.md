# Калькулятор с GUI на C++

Это приложение представляет собой калькулятор с графическим интерфейсом, написанный на чистом C++ с использованием библиотеки FLTK.

## Файлы проекта

- `calculator.cpp` - исходный код калькулятора
- `Makefile` - файл для компиляции программы
- `calculator` - исполняемый файл

## Компиляция

Для компиляции проекта выполните команду:

```bash
make
```

## Запуск

Для запуска калькулятора с графическим интерфейсом:

```bash
./calculator
```

**Важно:** Для работы GUI требуется X-сервер. В терминале без графического интерфейса приложение завершится с ошибкой.

## Функциональность

- Поддержка основных арифметических операций: сложение, вычитание, умножение, деление
- Работа с десятичными числами
- Кнопка смены знака (±)
- Кнопка очистки (C)
- Обработка ошибок (например, деление на ноль)
